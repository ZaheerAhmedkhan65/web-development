    JavaScript conditions are used to perform different actions based on different conditions. In this example, we use if and else statements to perform different actions based on different conditions.
  




  
Example 1

  

    In this example, we use if and else statements to perform different actions based on different conditions.
    


    
    let marks = 30;

    if (marks >= 33) {
      document.write("You have passed the exam");
    } else {
      document.write("You have failed the exam");
    }


    marks = 85;

    if (marks >= 80) {
      document.write("You have passed the exam with A+ grade");
    } else if (marks >= 60) {
      document.write("You have passed the exam with first B ");
    } else if (marks >= 45) {
      document.write("You have passed the exam with second C");
    } else if (marks >= 33) {
      document.write("You have passed the exam with third E");
    } else {
      document.write("You have failed the exam");
    }
