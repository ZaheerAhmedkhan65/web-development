JavaScript Recap - Variables, Conditions, Operators, Loops

1. Variables

Definition: Variables store data that can be used and modified throughout your program.

Types of Variables:

  • var - Function-scoped or globally-scoped, can be redeclared and updated.
  • let - Block-scoped, can be updated but not redeclared in the same scope.
  • const - Block-scoped, cannot be updated or redeclared.

Example:

var age = 25;
let name = "Alice";
const birthYear = 1995;

Remember: Use let and const instead of var to avoid scope-related issues.

2. Operators

Definition: Operators are used to perform operations on variables and values.

Types of Operators:

  • Arithmetic Operators: + - * / % (e.g., 5 + 3 returns 8)
  • Assignment Operators: = += -= *= /= (e.g., x += 5 is shorthand for x = x + 5)
  • Comparison Operators: == != === !== > < >= <= (e.g., 5 == "5" returns true, but 5 === "5" returns false)
  • Logical Operators: && || ! (e.g., true && false returns false)

Example:

let x = 10;
let y = 5;
console.log(x + y);  // Outputs: 15
console.log(x > y);  // Outputs: true

3. Conditional Statements

Definition: Conditional statements allow you to perform different actions based on different conditions.

Types:

  • if statement
  • if...else statement
  • else if ladder
  • switch statement

Example of if...else:

let age = 18;

if (age >= 18) {
    console.log("You are an adult.");
} else {
    console.log("You are a minor.");
}

Example of switch:

let day = 2;

switch (day) {
    case 1:
        console.log("Monday");
        break;
    case 2:
        console.log("Tuesday");
        break;
    default:
        console.log("Other day");
}

4. Loops

Definition: Loops allow you to run the same code multiple times with different values.

Types of Loops:

  • for loop
  • while loop
  • do...while loop
  • for...of and for...in loops

Example of for loop:

for (let i = 0; i < 5; i++) {
    console.log(i);
}
// Outputs: 0 1 2 3 4

Example of while loop:

let i = 0;

while (i < 5) {
    console.log(i);
    i++;
}
// Outputs: 0 1 2 3 4

Use for...in for looping over object properties and for...of for looping over iterable objects like arrays.
