2. Basics of Flexbox

Concept: Flexbox is a layout model that allows you to design complex layouts with ease by distributing space along a single axis.

  • Example Layout using flex

Summary

  • Flexbox Basics: Using display: flex to create flexible containers.
  • Flex Direction: Changing the direction of flex items.
  • Justify Content: Aligning items along the main axis.
  • Align Items: Aligning items along the cross axis.
  • Flex Wrap: Wrapping flex items onto multiple lines.
  • Align Self: Overriding alignment for individual items.
